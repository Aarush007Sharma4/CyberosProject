# CyberOS Nexus

CyberOS Nexus is a futuristic cyber defence simulation built using C++ with an optional HTML dashboard overlay.

This project simulates:
- authentication systems
- brute force prevention
- phishing attacks
- malware detection
- DDoS simulations
- digital forensics
- SOC monitoring

It was created as a first-year B.Tech C++ project.

---

## Files

- `CyberOS_Nexus.cpp`
- `CyberOS_Nexus.exe`
- `CyberOS_Nexus_Imagination.html`
