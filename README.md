CyberOS Nexus is a basic terminal-based cyber defence simulator built using C++.

The project simulates:
- Authentication systems
- Brute force prevention
- Phishing detection
- DDoS simulations
- Malware response
- Digital forensics
- SOC monitoring
- Admin defence controls

This was created as a first-year B.Tech C++ project inspired by Linux terminals and real Security Operations Centers (SOC).

# Running the Project

## Linux / WSL / macOS

### Install g++

Ubuntu / Debian:
```bash
sudo apt install g++
```

Arch Linux:
```bash
sudo pacman -S gcc
```

Fedora:
```bash
sudo dnf install gcc-c++
```

### Compile
```bash
g++ -std=c++17 -o CyberOS_Nexus CyberOS_Nexus.cpp
```

### Run
```bash
./CyberOS_Nexus
```

---

## Windows

### Compile
```bash
g++ -std=c++17 -o CyberOS_Nexus.exe CyberOS_Nexus.cpp
```

### Run
```bash
CyberOS_Nexus.exe
```

Or directly open:
```txt
CyberOS_Nexus.exe
```
