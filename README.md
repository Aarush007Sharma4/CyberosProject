# CyberOS Nexus

CyberOS Nexus is a futuristic cyber defence simulation built using C++ and a companion HTML interface.

The project was created as a first-year B.Tech C++ project to explore:
- authentication systems
- brute force prevention
- threat monitoring
- digital forensics
- incident response
- Linux-style terminal interaction

It combines a terminal-based backend with an optional web overlay for users who prefer a graphical interface.

---

## Project Overview

CyberOS Nexus simulates the behaviour of a modern Security Operations Center (SOC).

The system includes:
- authentication and lockout systems
- brute force attack simulations
- phishing and malware demonstrations
- threat intelligence dashboards
- forensic evidence analysis
- incident reporting
- Linux-style command console
- administrative containment tools

The project is educational and presentation-focused.

---

## Technologies Used

### Backend
- C++
- STL (maps, vectors, strings)
- File handling
- Threading and delays
- Console rendering

### Frontend Overlay
- HTML
- CSS
- JavaScript

## Features

### Authentication System

- brute force protection
- CAPTCHA triggering
- account lockout
- exponential backoff
- suspicious activity tracking

- brute force simulation
- phishing simulation
- DDoS simulation
- malware payload simulation
- spoofed IP behaviour

### Threat Intelligence
- live threat score
- radar simulation
- timeline events
- active incident monitoring

### Digital Forensics

- artifact analysis
- FNV-1a hashing
- incident chain tracking

### Administrative Controls
- network quarantine
- key rotation
- host isolation
- alert clearing
- full reset system

- report

## Running the C++ Version

### Compile

```bash
g++ -std=c++17 -o CyberOS_Nexus CyberOS_Nexus.cpp
```

### Run

```bash
./CyberOS_Nexus
```

---

## Running the HTML Overlay

If you prefer a graphical experience instead of the terminal version:

1. Open:
   `CyberOS_Nexus_Imagination.html`

2. Run it directly in a browser.

No installation or server is required.

The HTML overlay acts as a cinematic visual layer inspired by:

* Linux terminals
* SOC dashboards
* futuristic cyber defence systems
